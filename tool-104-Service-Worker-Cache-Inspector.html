Caching Strategies

Common Service Worker patterns.

Cache First
Checks cache before network. Excellent for static assets (images, fonts, stylesheets).
Network First
Tries network, falls back to cache on failure. Good for frequently updating data (API requests).
Stale-While-Revalidate
Returns cached response immediately if available, then fetches from network in background to update cache for next time.
